🏏 IPL Analysis Dashboard 

Project Overview

This project presents an interactive IPL (Indian Premier League) Analysis Dashboard till 2024 built using Power BI. The dashboard provides deep insights into team performance, player statistics, match outcomes, and seasonal trends across multiple IPL seasons.

The goal of this project is to transform raw cricket data into meaningful visual insights for better decision-making and analysis.

🎯 Objectives
Analyze IPL match data across seasons
Identify top-performing teams and players
Track win/loss patterns and trends
Provide interactive visualizations for better understanding

📊 Dashboard Features
📅 Season-wise Analysis – Performance comparison across different IPL seasons
🏆 Tournament Winner Insights – Year-wise winning teams
👥 Team Performance – Matches played, wins, losses, win percentage
🔥 Top Players Analysis – Highest run scorers, wicket takers
📈 Match Trends – Toss impact, venue analysis, match outcomes
🎯 Interactive Filters (Slicers) – Filter by season, team, venue, player

🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Excel / CSV Dataset – Data source
Data Cleaning & Transformation – Power Query

📂 Dataset Description
The dataset includes:

Match details (date, teams, venue, winner)
Deliviers details (runs, wickets, etc.)
Toss decisions and outcomes

⚙️ Key DAX Measures Used
Tournament Winner Calculation
Total Matches Played
Win Percentage
Player Performance Metrics

🚀 How to Use
Open the .pbix file in Power BI Desktop
Use slicers to filter data (season, team, etc.)
Explore visuals and insights interactively
